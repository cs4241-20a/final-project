Welcome New User

If you would like to login via github, please select the login via github button. If we were to add on to our application, this would allow you to join with your github username.

To see your own video, please select the “connect” button. This will allow you to see your own video in one of the video tiles below.

To join an existing call, type the name of the bubble corresponding to the call in the “join an old bubble” and press the plus sign. Currently, you will only be able to join “one”, “two”, or “three”. If we were to extend our application, we would implement the ability to join any of the bubbles shown above the text entry boxes by typing their name into the box.

To create and join a new bubble, please enter the name of your new call in the “new bubble” box and press the plus sign.

To pause the chrome tools debugger, select the “Debug” button with chrome devtools open.
